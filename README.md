# mini_Gemini
This project is a modern AI chat web application built using Flask (Python) and the Google Gemini API. It provides a clean and interactive user interface that allows users to communicate with an AI model in real time.

The application includes features such as sending messages, receiving AI-generated responses, and managing multiple chat sessions with a user-friendly layout inspired by modern AI tools.

Additionally, the project explores core NLP concepts like tokenization, encoding, and decoding using Hugging Face models in Google Colab, giving deeper insight into how AI models process text internally.

Technologies used:
- Python (Flask)
- Google Gemini API
- HTML, CSS, JavaScript
- Hugging Face Transformers (for NLP concepts)
